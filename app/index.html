Disconnected

Open this page from the ESP32 (http://me424-main.local/) for the simplest setup. If you use http://localhost:8080/app/, set Robot host to the main board (saved in the browser). Add ?usb=1 to force USB serial.

Manual Control

Zero / Home

Use Auto home under each axis for firmware homing: home s1 soft-zeros the turntable; home s2–home s4 seek limits then zero; home s5 seeks the Hall zero (sensor GPIO14 / HW-477; limits shows S5H=0 at zero) then zeros C5. Stage 5 Hall does not block motion—only used for homing.
