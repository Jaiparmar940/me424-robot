MOSFET / attachments

Sends the same commands as the console (mag on, vac off, …) to the main board; the main forwards to the MOSFET slave. Saw now uses only slave GPIO 22 for BLDC controller input: saw off forces 0V and saw on applies PWM from saw speed 0–100. Button highlights follow the slave STATUS; speed syncs from SAW_SPD / ACK SAW SPEED.

Zero / Home

Limit autohome bounce-homes all five stages once (S1, S3, S4, S2, S5) with safe pre-positioning. Lift autohome does a full two-pass calibration (straight then bounce) for all stages, then moves to the vertical pose and sets it as the tracked origin (setpos 0…0). Both use Jog Speed / Ramp Steps for syncabs moves.

Use Auto home under each axis for individual firmware homing (bounce). Stage 5 uses the Hall sensor (GPIO14 / HW-477; limits shows S5H=0 at zero). Stage 5 Hall does not block motion—only used for homing.

Logical zero offset (steps, S1–S5): use after the arm moves to re-align the tool frame. Apply / Clear save to NVS then run a coordinated move (syncabs @ 800 stp/s, ramp 200) so the same logical pose is held in space. Current pose → logical 0 only updates offsets (no move). S2 is the lift pair (C1+C4). S5 tool tweak jogs the wrist 800 steps at 200 stp/s for mating tests.
