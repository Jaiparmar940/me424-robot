Manual Control

MOSFET / attachments

Sends the same commands as the console (mag on, vac off, …) to the main board; the main forwards to the MOSFET slave. Saw power is GPIO 23 on the slave; BLDC throttle is PWM on slave GPIO 22 (saw speed 0–100). PWM is applied only while saw on. Button highlights follow the slave STATUS; speed syncs from SAW_SPD / ACK SAW SPEED.

Zero / Home

Go to vertical runs one syncabs to [7533, −6027, −3874, 7533, 0, −2847] (VERTICAL_POSE in the app script). Lift autohome ends at vertical after home s1 bounce, then setpos 0…0 so that pose is the tracked origin. Uses Jog Speed / Ramp Steps.

Use Auto home under each axis for firmware homing: home s1 jogs C6 + to the S1 limit then zeros C6; home s2–home s4 seek limits then zero; home s5 seeks the Hall zero (sensor GPIO14 / HW-477; limits shows S5H=0 at zero) then zeros C5. Stage 5 Hall does not block motion—only used for homing.
